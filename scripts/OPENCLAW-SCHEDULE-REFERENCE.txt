Telegram on failure (watchthisapp commands only)
-----------------------------------------------
1. Create a bot with @BotFather; get the bot token.
2. Get your chat_id (e.g. message @userinfobot or your bot and read updates via getUpdates).
3. Export before Hermes runs (or put in ~/.hermes/.env if Hermes injects env):
     export TELEGRAM_BOT_TOKEN="..."
     export TELEGRAM_CHAT_ID="..."
4. Point Hermes skill shell steps at the wrapper, e.g.:
     /path/to/watchthisapp/scripts/run-with-telegram-alert.sh "Vega GSC Daily" \
       zsh -c 'cd "$HOME/projects/watchthisapp" && npm run cron:gsc'

For skills outside this repo (job-hunter, fursbliss-*), wrap the same way around
your existing command once TELEGRAM_* is set.
