Hermes restore — one command
=============================

If `./scripts/restore-hermes-from-timemachine.sh` is missing, sync the repo first:

  cd ~/projects/watchthisapp && git pull origin main

Mount your Time Machine volume, then:

  ./scripts/restore-hermes-from-timemachine.sh

Defaults (edit the script if your snapshot path changes) are inside
`scripts/restore-hermes-from-timemachine.sh`.

Restore and push `~/.hermes` to the Mac mini in one go:

  MINI_HOST=openclaw@your-mini.local ./scripts/restore-hermes-from-timemachine.sh
