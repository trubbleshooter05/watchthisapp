# Daily Growth Checklist

## Daily (15-30 min)
- Check pending submissions in `growth/backlinks.csv` and update status.
- Submit to 1-2 new backlink targets.
- Post one high-value community comment (Reddit/Quora/Indie Hackers).
- Track one new referral source in analytics.
- Log wins/blockers in notes column.

## Weekly
- Ship one new comparison/SEO page worth linking to.
- Follow up on pending submissions from Day 2/7/14 cadence.
- Refresh TikTok content queue (3-5 clips scripted).
- Review top-converting referral links.
- Prune low-value targets and replace with better ones.

## Follow-Up Cadence
- **Day 0:** Submit / initial outreach
- **Day 2:** Friendly follow-up ("wanted to bump this")
- **Day 7:** Add proof point (traffic/user feedback)
- **Day 14:** Final check-in, then mark archived if no reply

## Rejection Playbook
1. Capture exact rejection reason in `notes`.
2. Update positioning copy (length/category/tone) if needed.
3. Re-submit once with improved copy if allowed.
4. If not allowed, move to next target immediately.
5. Convert rejection into content idea (e.g., "Alternative to X" page).
